# AI Usage

This project was built with AI assistance during the Band of Agents Hackathon.

AI was used for:

- product strategy and judging-rubric alignment
- competitive positioning research and submission framing
- drafting deterministic recall workflow code
- drafting source evidence parsing, citation display, traceability math, and approval receipt code
- drafting key-gated Featherless and AI/ML API adapter paths
- drafting the FastAPI proof, regulator dispatch, e-signature, and enterprise dry-run surfaces
- drafting frontend UI code for `/console`, `/demo/judge`, `/submission`, `/proof`, and `/docs`
- generating tests, smoke checks, and verification plans
- preparing README and submission copy

Human-controlled work:

- final product direction
- credential setup for external services
- approval of partner-provider keys or live external AI calls
- approval of any real regulator submission, SAP/Oracle tenant write, package publish, or hackathon submission
- final demo video recording and public submission decisions

No final hackathon submission is made without explicit human approval.
